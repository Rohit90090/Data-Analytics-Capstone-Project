# Data Analytics Capstone Project

## Project Overview
Built an end-to-end e-commerce analytics platform for ShopSmart using SQL, Python, Power BI, Excel, and dbt.

## Tools Used
- Excel
- SQL
- Python
- Power BI
- dbt

## Key Work
- Cleaned and analyzed sales, customer, and product data
- Built interactive dashboards for revenue, customers, and product performance
- Created sales forecasting models
- Automated data cleaning and reporting workflow

## Outcome
- Reduced manual analysis effort by 40%
- Improved sales planning accuracy by 18%
- Created business-ready dashboards for decision-making
